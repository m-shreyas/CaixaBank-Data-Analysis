# 📊 Banking Financial Performance Dashboard (2010)

An executive-level financial analytics project built using **Tableau Public**, focusing on banking transaction trends, customer behavior, and KPI storytelling.

---

## 🚀 Project Overview

This dashboard analyzes monthly banking performance data for 2010 to uncover:
- Spending trends
- Transaction behavior
- Customer engagement patterns
- Executive KPIs

The project emphasizes **business storytelling**, not just visualization.

---

## 📌 Key KPIs

| Metric | Value |
|------|------|
| Total Spend | $16.91M |
| Avg Transaction Value | $41.88 |
| Max Active Customers | 1,108 |
| Max Active Cards | 2,603 |

---

## 📈 Dashboard Insights

### 🔹 Monthly Spend Trend
- Growth from January to April
- Sharp decline in May

### 🔹 Transaction Volume
- Volume drop is the main driver of revenue decline
- Not caused by lower transaction value

### 🔹 Customer Behavior
- Active customers increased steadily
- Engagement frequency decreased

---

## 🧠 Business Interpretation

- Strong customer retention
- Declining transaction frequency
- Potential seasonal or behavioral impact

---

## ✅ Recommendations

- Investigate May transaction decline
- Launch engagement-focused initiatives
- Track transaction volume more frequently
